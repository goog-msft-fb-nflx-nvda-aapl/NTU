Abstract

3D Gaussian Splatting (3DGS) achieves state-of-the-art novel view synthesis quality and real-time rendering speed, but its adoption is limited by a fragmented setup process requiring COLMAP, custom CUDA extensions, and manual pipeline orchestration. GaussianOne wraps the full 3DGS pipeline — from raw images through COLMAP reconstruction, Gaussian optimization, novel-view rendering, and quantitative evaluation — into a single shell command. We validate on four standard scenes from the Tanks & Temples and Deep Blending benchmarks, achieving competitive PSNR scores of 25.88–37.25 dB.

Pipeline

One command drives the entire workflow:

# Usage
bash gaussianone.sh <input_images_folder> <output_name>

# Example
bash gaussianone.sh ./my_photos/ my_scene
