Final Results

What I Improved

Additional engineering work outside the designated TODO blocks, all marked with #NewFeature.

Implementation changes

  • Headless mode via --headless to run simulations without GUI windows on remote servers.
  • Auto-exit on finish so the simulator terminates automatically when the track is completed.
  • Timeout mechanism via --max_time to stop unstable or oscillatory runs during tuning.
  • CLI gain overrides for PID gains and LQR weights, enabling fast parameter sweeps without editing source code.

Tuning workflow

  • Built an automated grid search script to evaluate controller settings across multiple runs.
  • Tested multiple lateral controllers, including PID, Pure Pursuit, Stanley, and LQR.
  • Selected Stanley with kp = 8.0 as the final submission due to its low CTE and robust behavior across all three tracks.
